# 🐍 Python Mini-Projects Collection

Welcome to my Python sandbox! This repository is a growing collection of beginner-to-intermediate Python projects. I am building these to strengthen my understanding of core programming concepts like control flow, functions, loops, data structures, and modular programming.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Libraries Used:** `random`, `string`

---

## 📂 Project Directory

### 1. Guess the Number (1-100)
A classic interactive CLI (Command Line Interface) game where the computer selects a random number between 1 and 100, and the player has to guess it in the fewest attempts possible.
* **Concepts Explored:** `while` loops, conditional statements (`if/else`), user input validation, and attempt tracking.
* **File:** `guess_the_number.py`

### 2. Stone, Paper, Scissors
A digital version of the classic hand game. The player enters their choice, and the computer randomly selects its move from a predefined list. The script then evaluates the winner based on the game's traditional rules.
* **Concepts Explored:** Functions (`def`), dictionaries, lists, and complex `if/elif/else` logic.
* **File:** `stone_paper_scissors.py`

### 3. Hangman Game
A fully functional terminal-based Hangman game. The program randomly selects a valid word from a custom list, and the player must guess it letter by letter before their lives run out. Features dynamic ASCII art to visually represent the hangman!
* **Concepts Explored:** Sets (for tracking guessed letters), list comprehensions, importing custom external modules/files, and advanced string manipulation.
* **Files Needed:** `hangman.py`, `words.py` (contains the word list), and `hangman_visual.py` (contains the ASCII art).
