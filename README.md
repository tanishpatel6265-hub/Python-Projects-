## 📂 Project Directory

### 1. Guess the Number (1-100)
A classic interactive CLI (Command Line Interface) game where the computer selects a random number between 1 and 100, and the player has to guess it in the fewest attempts possible.
* **Concepts Explored:** `while` loops, conditional statements (`if/else`), user input validation, and attempt tracking.
* **File:** `guess_the_number.py` *(Update with your actual file name)*

### 2. Stone, Paper, Scissors
A digital version of the classic hand game. The player enters their choice, and the computer randomly selects its move from a predefined list. The script then evaluates the winner based on the game's traditional rules.
* **Concepts Explored:** Functions (`def`), dictionaries, lists, and complex `if/elif/else` logic.
* **File:** `stone_paper_scissors.py` *(Update with your actual file name)*
